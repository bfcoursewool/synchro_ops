To install the Synchro App dev environment: <br>

git clone git@github.com:bfcoursewool/synchro_ops.git <br />
cd synchro_ops <br />
git submodule init <br />
git submodule update <br />

install vagrant then: <br />

vagrant up <br />
vagrant ssh <br />
